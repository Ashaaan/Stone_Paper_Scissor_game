# Rock Paper Scissors - Modern Web Edition

A sleek, responsive, and animated Rock Paper Scissors game built with Modern JavaScript and Tailwind CSS v4. This project features a dynamic scoring system, a custom-built computer AI, and glowing glassmorphism UI effects.

## üïπÔ∏è How to Play

Choose your weapon by clicking on the Rock, Paper, or Scissor circular images.

The computer will simultaneously make a random choice.

The result will appear in a glowing box below the score:

Green Glow: You won! Your score increases.

Red/Pink Glow: Computer won. Computer score increases.

Blue Glow: It's a draw!

Track your progress in the live score counters.
